ruby-candle

candle - Minimalist ML framework - for Ruby

Usage

require "candle"

x = Candle::Tensor.new([1, 2, 3, 4, 5, 6])
x = x.reshape([3, 2])
# [[1., 2.],
#  [3., 4.],
#  [5., 6.]]
# Tensor[[3, 2], f32]

Development

FORK IT!

git clone https://github.com/your_name/ruby-candle
cd ruby-candle
bundle
bundle exec rake compile

Implemented with Magnus, with reference to Polars Ruby

Policies - The less code, the better. - Ideally, the PyPO3 code should work as is. - Error handling is minimal.
